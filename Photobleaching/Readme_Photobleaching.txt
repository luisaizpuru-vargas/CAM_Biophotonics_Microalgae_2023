This readme file is a guide to understand the order of the folders of the photobleaching analysis. This data was used to construct the curves for Figures 1 and 2 the paper: "Photosynthetic Microalgae-Enhanced Oxygenation Amplifies Vascular Response to Photodynamic Therapy in a Chorioallantoic Membrane Model"

The GitHub repository contains 7 folders. Each folder contains the raw data to construct each curve.
